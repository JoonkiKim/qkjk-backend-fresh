[MySQL에 대해 알아보자]

접속명령어 
뒤에 항상 세미콜론; 붙여주기

mysql -u root -p;
u는 유저라는 뜻. root는 관리자 계정으로 들어간다는 뜻. p는 패스워드

show databases;
데이터 베이스를 보여준다

use mysql;
mysql이라는 데이터베이스를 사용하겠다는 뜻

show tables; 
테이블을 보여줘

테이블 조회하고 나서는 SQL구문으로 각 테이블을 조회한다
select * from user;
근데 이렇게 하면 터미널에서 잘 안보이기 때문에 DBEAVER를 사용한다!

mysql을 항상 켜고 디비버를 켜야된다
